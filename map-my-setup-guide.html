How it ranks

What it flags first

It ranks files by how often they load, not by how long they are. A file Claude reads on every turn matters more than a long file you rarely open.

Install

Two commands

Type these into a Claude Code session.

1. Add the library (once)
/plugin marketplace add aistudycamp/skills
2. Install the skill
/plugin install map-my-setup@aistudycamp-skills
Use it

Say "map my setup"

It scans your setup without changing anything, then opens the dashboard in your browser. Run /mcp first and paste the result so it catches every connector.
