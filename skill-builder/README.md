# Skill Builder

Build your first Claude Code skill in about 45 minutes.

## What you'll build

A reusable skill - a prompt that Claude loads automatically when you need it. Like a recipe card for a workflow you repeat often.

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/aistudycamp/skills.git
   ```

2. cd into skill-builder:
   ```bash
   cd skills/skill-builder
   ```

3. Run Claude:
   ```bash
   claude
   ```

4. **Say hello!** - Claude won't start automatically, type "hello" or any greeting to begin.

## What's inside

- `CLAUDE.md` - your interactive instructor (Claude reads this automatically)
- `examples/` - 3 example skills to reference
