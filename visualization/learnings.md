# Visualization Skill — Learnings

This file accumulates lessons from real usage. After a visualization is built, if you corrected something or stated a preference, a note gets added here. Over time the skill learns your style.

It starts empty — that's expected. Entries get added as you use the skill.

---

Format for each entry:

```
## Learning: <short title> (YYYY-MM-DD)
**What happened:** <the situation>
**Lesson:** <what to do differently next time>
**Apply when:** <when future runs should use this>
```
